Виртуальный аудио-кабель

VB-Cable позволяет приложению слышать звук из Zoom без конфликта с микрофоном.

⏳ Проверяю...

Установка (бесплатно, 2 минуты):

  1. Скачайте VB-Cable по ссылке ниже
  2. Запустите VBCABLE_Setup_x64.exe от администратора
  3. Перезагрузите компьютер
  4. В Zoom: Настройки → Аудио → Динамик → CABLE Input
  5. Нажмите «Проверить снова»
