# LessonLive

**Real-time транскрипция для онлайн-занятий** — приложение которое слушает вашу речь и речь собеседника во время урока через Zoom или Google Meet, автоматически переводит её в текст и сохраняет транскрипцию занятия.

Поддерживает французский, русский и английский языки одновременно. Работает полностью локально — ваши данные никуда не отправляются.

## Установка

### Шаг 1 — Установите Python

Скачайте Python 3.10 или 3.11 с официального сайта: https://www.python.org/downloads/

При установке обязательно поставьте галочку **"Add Python to PATH"**.

Проверьте установку в командной строке:
```
python --version
```
Должно показать `Python 3.10.x` или `Python 3.11.x`.

### Шаг 2 — Скачайте LessonLive

Скачайте репозиторий с GitHub:
```
git clone https://github.com/ta0032/LessonLive.git
```
Или скачайте ZIP архив через кнопку **Code → Download ZIP** и распакуйте в удобную папку.

### Шаг 3 — Настройте Windows Audio

Перед запуском установки необходимо настроить захват звука. Без этого приложение не сможет слышать голос собеседника.

Подробная инструкция в разделе [Настройка Windows Audio](#настройка-windows-audio) ниже.

### Шаг 4 — Зарегистрируйтесь на HuggingFace

Для скачивания модели разделения спикеров нужен бесплатный токен. Инструкция в разделе [Регистрация на HuggingFace](#регистрация-на-huggingface) ниже.

### Шаг 5 — Запустите установку

Дважды кликните на файл `launch.pyw` в папке LessonLive.

Если приложение запускается впервые — автоматически откроется окно установки с логами. Установка включает:

- Проверку Python и GPU
- Установку всех необходимых библиотек
- Скачивание модели Whisper (500 MB — 3 GB в зависимости от выбора)
- Скачивание модели pyannote для разделения спикеров (~1.5 GB)

**Установка занимает 10–30 минут** в зависимости от скорости интернета.

В процессе установки вам потребуется:
- Выбрать размер модели Whisper (рекомендуется `small` для начала)
- Вставить токен HuggingFace

После завершения установки приложение запустится автоматически.

## Настройка Windows Audio

Приложение использует **Стерео микшер** (Stereo Mix) для захвата звука из Zoom и Google Meet. Это встроенная функция Windows которая позволяет записывать всё что воспроизводится через динамики.

### Включение Стерео микшера

1. Нажмите правую кнопку на иконке звука в правом нижнем углу панели задач
2. Выберите **Параметры звука**
3. Прокрутите вниз и нажмите **Дополнительные параметры звука**
4. Перейдите на вкладку **Запись**
5. Нажмите правую кнопку мыши в пустом месте списка
6. Выберите **Показать отключённые устройства**
7. Найдите **Стерео микшер** → нажмите правую кнопку → **Включить**
8. Нажмите **ОК**

Если Стерео микшер отсутствует — ваша звуковая карта его не поддерживает. В этом случае установите **VB-Cable** (бесплатно): https://vb-audio.com/Cable/

### Важные настройки Стерео микшера

Чтобы избежать эффекта эха при записи:

1. Откройте вкладку **Запись** в параметрах звука
2. Дважды кликните на **Стерео микшер**
3. Перейдите на вкладку **Прослушать**
4. Убедитесь что галочка **"Прослушивать с данного устройства"** **снята**

### Настройка вывода звука

Стерео микшер захватывает звук с того устройства вывода которое активно в данный момент. Для корректной работы:

- Во время занятия звук из Zoom или Google Meet должен воспроизводиться через **динамики ноутбука** или **динамики ПК**
- Если вы используете наушники через Bluetooth — звук может не захватываться
- Рекомендуется использовать проводные наушники или работать без наушников

## Регистрация на HuggingFace

HuggingFace — платформа для AI-моделей. Регистрация бесплатная и нужна только для однократного скачивания модели разделения спикеров.

### Шаг 1 — Создайте аккаунт

Перейдите на https://huggingface.co/join и зарегистрируйтесь.

### Шаг 2 — Получите токен

1. Войдите в аккаунт
2. Перейдите в настройки: https://huggingface.co/settings/tokens
3. Нажмите **New token**
4. Введите любое название (например `LessonLive`)
5. Тип токена выберите **Read**
6. Нажмите **Generate a token**
7. Скопируйте токен — он начинается с `hf_`

### Шаг 3 — Примите лицензию модели

Перейдите на страницу модели и нажмите **"Agree and access repository"**:

- https://huggingface.co/pyannote/speaker-diarization-3.1
- https://huggingface.co/pyannote/segmentation-3.0

### Важно

После скачивания модели токен больше не нужен для работы приложения. Модель хранится локально в папке `models/pyannote/`.

## Запуск приложения

### Первый запуск

Дважды кликните на файл **`launch.pyw`** в папке LessonLive.

При первом запуске откроется окно установки. После завершения установки приложение запустится автоматически.

### Последующие запуски

Дважды кликните на **`launch.pyw`**.

Откроется окно с логами загрузки. Когда все модели загрузятся — окно закроется автоматически и откроется браузер с приложением.

Время загрузки: **30–60 секунд** (зависит от скорости GPU).

### Автозагрузка при старте Windows

Чтобы LessonLive запускался автоматически при входе в Windows:

1. Нажмите `Win + R`, введите `shell:startup`, нажмите Enter
2. Создайте ярлык файла `launch.pyw` в открывшейся папке

## Работа с приложением

### Интерфейс

Приложение открывается в браузере по адресу `http://localhost:8000`.

**Шапка** содержит:
- Название приложения LessonLive
- Индикатор записи (красный пульсирующий кружок во время записи)
- Статус моделей (загружаются / готово к работе)
- Переключатель языка интерфейса RU/EN

**Левая панель** содержит:
- Поля для имён спикеров — введите своё имя и имя ученика, нажмите **Сохранить**
- Список прошлых занятий — нажмите на занятие чтобы открыть его текст

**Область транскрипции** — основная рабочая зона. Здесь в реальном времени появляется текст с разделением по спикерам. Ваша речь отображается синим, речь ученика — зелёным.

**Нижняя панель** содержит:
- Кнопку **Начать запись** / **Остановить**
- Кнопку **Очистить** — очищает экран (файл не удаляется)
- Счётчик фраз текущей сессии

### Проведение занятия

1. Откройте Zoom или Google Meet и начните звонок
2. Убедитесь что звук воспроизводится через динамики (не наушники)
3. Откройте LessonLive в браузере
4. Дождитесь статуса **"Готово к работе"**
5. Нажмите **Начать запись**
6. Говорите — ваша речь появляется сразу
7. Речь ученика появляется с небольшой задержкой (~3 секунды)
8. По окончании нажмите **Остановить**

Транскрипция автоматически сохраняется в папку `sessions/` в формате `.txt`.

### Сохранённые занятия

Все занятия сохраняются в папке `LessonLive/sessions/` с именем по дате и времени, например `2024-03-14_22-30_lesson.txt`.

В конце каждого файла — статистика: длительность занятия и количество фраз по каждому спикеру.

## Иконка в трее

После запуска в правом нижнем углу экрана (системный трей) появляется иконка LessonLive.

### Состояния иконки

**Синяя иконка** — сервер запущен и работает. Приложение готово к работе.

**Серая иконка** — сервер остановлен или не отвечает.

Иконка обновляет своё состояние каждые 3 секунды.

### Меню иконки

Нажмите правую кнопку на иконке (или левую — зависит от системы):

**Открыть LessonLive** — открывает приложение в браузере. Если сервер был остановлен — перезапускает его.

**Выйти** — останавливает сервер и полностью закрывает приложение.

### Закрытие окна с логами

При запуске появляется окно с логами загрузки. Оно закрывается автоматически когда все модели загружены и браузер открылся.

Если вы закроете это окно крестиком вручную — **сервер продолжает работать** в фоне. Приложение остаётся доступным через иконку в трее.

### Повторный запуск

Если запустить `launch.pyw` когда приложение уже работает — новое окно не откроется. Вместо этого просто откроется браузер с уже запущенным приложением.

## Частые вопросы

**Не слышно голос ученика в транскрипции**

Проверьте что Стерео микшер включён (раздел [Настройка Windows Audio](#настройка-windows-audio)). Убедитесь что звук из Zoom воспроизводится через динамики, а не через Bluetooth наушники.

**Появляются фразы "Thank you", "You" которые никто не говорил**

Это галлюцинации модели Whisper на тихих участках. Попробуйте говорить чуть громче или поднести микрофон ближе.

**Кнопка "Начать запись" неактивна**

Дождитесь статуса "Готово к работе" в шапке. Модели загружаются 30–60 секунд после запуска.

**Приложение не открывается в браузере**

Нажмите на иконку LessonLive в трее → **Открыть LessonLive**.

**Как переустановить или обновить модели**

Запустите `setup.py` через терминал: `python setup.py`

**Где хранятся транскрипции занятий**

В папке `LessonLive/sessions/`. Файлы в формате `.txt`, названы по дате и времени занятия.

**Можно ли использовать на Mac**

Приложение разработано для Windows. На Mac потребуется заменить Стерео микшер на аналогичное решение (например BlackHole) и может потребоваться настройка аудио устройств.

## Структура проекта

```
LessonLive/
├── launch.pyw          — точка входа, двойной клик для запуска
├── app.py              — FastAPI сервер
├── setup.py            — установка зависимостей и моделей
├── state.py            — глобальное состояние приложения
├── requirements.txt    — список Python зависимостей
├── core/
│   ├── audio.py        — захват аудио (микрофон + Стерео микшер)
│   ├── transcribe.py   — транскрипция через Whisper
│   ├── diarize.py      — разделение спикеров через pyannote
│   └── session.py      — управление сессией и сохранение
├── server/
│   ├── routes.py       — HTTP маршруты
│   └── socket.py       — WebSocket для обновлений в реальном времени
├── static/
│   ├── index.html      — главная страница
│   ├── setup.html      — страница онбординга
│   ├── app.js          — клиентская логика
│   └── style.css       — стили
├── models/             — AI модели (создаётся при установке)
└── sessions/           — сохранённые транскрипции
```

## Технологии

- **faster-whisper** — транскрипция речи в текст

- **FastAPI** — веб-сервер
- **sounddevice** — захват аудио
- **tkinter** — окно запуска и установки
- **pystray** — иконка в системном трее
